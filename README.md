A body-coupled communication (BCC) PoC video demonstration, including:\
• Static scene test.mp4\
• Video streaming test.mp4\
• Multi-posture test.mp4

The first video shows the performance of the BCC system in a static scenario with the subject keeping a stationary posture. The flashing frequency of the LED at the Rx indicates the quality of communication, and the faster it is, the smoother the link. The screen equipped in Rx can display real-time packet error rate (PER).

The second video presents the video transmission application based on the BCC link, which verifies the feasibility and practicality of the proposed scheme in multimedia service scenarios.

The third video presents the performance of the BCC link with variant postures.
